# Doodle Pad

A browser-based drawing canvas with dynamic brushes. Drawing speed controls both stroke thickness and color brightness — draw slowly for thick, vivid strokes; draw fast for thin, bright ones.

## Features

- **Speed-reactive brush** — velocity changes line width and lightness in real time
- **Color swatches** — six palette options including a white/neutral mode
- **Adjustable size** — slider to set the base brush size
- **Touch support** — works on mobile and tablet
- **Save as PNG** — export your drawing with one click

## Usage

Open `index.html` in any modern browser. No build step or dependencies required.
